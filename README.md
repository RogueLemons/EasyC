# The Crucible C Project

> “The good news about computers is that they do what you tell them to do. The bad news is that they do what you tell them to do.”
>
> — Ted Nelson

The Crucible C Project is a C ecosystem designed to make systems programming easier to read, write, and reason about through structured safety layers and clearer abstractions that promote clean, predictable, and safer code.

Includes:
- [AlloyCTranspiler](AlloyCTranspiler_Prototype): adds extra keywords to C for safety and readability
- [IronCLib](IronCLib): a quick-and-easy drop-in header library for safer C code
- [SteelCLib (upcoming)](SteelCLib): a bigger, better version of IronCLib with more flexibility at complexity cost
- [WorkshopC (upcoming)](WorkshopC): a code parser for safer and clearer code to follow a mold of coding
